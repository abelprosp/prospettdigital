# Fonte Classy Marisa

Para usar a fonte "Classy Marisa" no projeto, adicione os arquivos da fonte nesta pasta (`public/fonts/`).

## Arquivos necessários:

- `ClassyMarisa-Regular.woff2` (recomendado)
- `ClassyMarisa-Regular.woff` (fallback)
- `ClassyMarisa-Regular.ttf` (fallback adicional)
- `ClassyMarisa-Bold.woff2` (recomendado)
- `ClassyMarisa-Bold.woff` (fallback)
- `ClassyMarisa-Bold.ttf` (fallback adicional)

## Onde obter:

A fonte "Classy Marisa" pode ser adquirida em:
- Creative Market: https://creativemarket.com
- Outras plataformas de fontes premium

## Após adicionar os arquivos:

A fonte será automaticamente carregada pelo CSS global. Não é necessário fazer mais nada!
